A comprehensive usability study evaluating and redesigning the Newegg website through heuristic evaluation and user testing with 12 participants

Project Overview

Conducted comprehensive usability testing with 12 participants to identify key pain points in Newegg's e-commerce platform. Through heuristic evaluation and user testing, I created an interactive Figma prototype that significantly improved user experience and task completion efficiency.

Design Solution & Process

Redesign Strategy

Focused on creating a more intuitive and accessible shopping experience through improved navigation, enhanced filtering, and better visual hierarchy.

🧭 Simplified Navigation

Redesigned navigation structure with clear categories, improved menu organization, and intuitive breadcrumbs

🔍 Enhanced Search & Filtering

Implemented advanced search with autocomplete, improved filtering options, and better result display

🎨 Improved Visual Hierarchy

Applied consistent typography, better color contrast, and cleaner product listings for improved readability

♿ Accessibility Compliance

Ensured WCAG compliance with proper keyboard navigation, color contrast, and screen reader support
